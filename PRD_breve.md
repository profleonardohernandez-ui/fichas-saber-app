# PRD breve · Fichas de estudio pruebas Saber

## Propósito
Construir una SPA ligera para estudiar preguntas tipo Saber con enfoque pedagógico: pregunta guía, caso resumido, despliegue de pregunta original, respuesta y retroalimentación.

## Usuarios
- Docente: proyección y análisis en clase.
- Estudiante: estudio autónomo y repaso.

## Requisitos base
- Una ficha por pantalla.
- Filtros por tema, grado y año.
- Pregunta original desplegable.
- Opciones clicables.
- Retroalimentación inmediata.
- Persistencia local de respuestas.
- Arquitectura modular: HTML, CSS, JS y base de datos separados.
